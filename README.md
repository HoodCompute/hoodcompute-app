# HoodCompute

Programmable wallets, spending policies, and on-chain transactions for autonomous AI agents. Built on Robinhood Chain.

## Overview

HoodCompute gives each AI agent its own sovereign wallet on Robinhood Chain, enforces spending rules before any transaction fires, and writes every transfer to the chain as a permanent, independently verifiable record. This repository contains the web application.

## Stack

- **Next.js 16** - App Router, Turbopack dev server, standalone output
- **React 19** with strict TypeScript
- **Tailwind CSS v4** with a custom oklch-based design token system
- **shadcn/ui** for accessible UI primitives
